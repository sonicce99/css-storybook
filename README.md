# Next-CSS-Storybook

[https://nextjs-css-storybook.vercel.app](https://nextjs-css-storybook.vercel.app)

## 프로젝트 설명

styled-component를 제외한 어떠한 외부 라이브러리 없이 오직 CSS만으로 자연스러운 인터렉션 구현을 목표로 합니다.

Storybook을 통해 다양한 UI 인터넥션을 볼 수 있습니다.
